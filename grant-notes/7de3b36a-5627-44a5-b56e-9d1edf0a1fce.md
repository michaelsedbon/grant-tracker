# NEB Prizes 2026 — Application Checklist (Category 3: Arts, Culture, Heritage)

**Deadline: March 22, 2026 (19:00 CET)**
**Strand: A — Champions (existing, completed projects)**
**Category: 3 — Arts, Culture, and Heritage as Drivers of Change**

---

## 1. Eligibility Requirements

- [ ] Individual or organisation applicant
- [ ] Project implemented in the EU (France ✓)
- [ ] Project is existing AND completed → shows positive results
- [ ] Application submitted by project owner or authorised representative
- [ ] No previous EU monetary prize for this project
- [ ] Project is replicable in other contexts
- [ ] Submission in English via online platform

## 3. NEB Core Values (40/100 points)

### 3a. Sustainability (environmental)
- [ ] Project reduces environmental impact
- [ ] Uses circular/eco-design approaches (reuse, bio-based materials, etc.)
- [ ] Nature-based solutions or biodiversity preservation
- [ ] Evidence of environmental sustainability claims

### 3b. Inclusion
- [ ] Accessible and affordable for all
- [ ] Celebrates diversity or addresses vulnerable communities
- [ ] Inclusive management / "design for all"
- [ ] Promotes social interaction across generations

### 3c. Aesthetics & Quality of Experience
- [ ] High design quality — connects to place
- [ ] Fosters sense of belonging
- [ ] Meaningful cultural/emotional experiences
- [ ] Integrates new cultural and social values

## 4. Working Principles (35/100 points)

### 4a. Participatory Process
- [ ] Communities involved in co-design
- [ ] Communities involved in decision-making
- [ ] Communities involved in implementation
- [ ] Evidence of empowerment of local communities

### 4b. Multi-level Engagement
- [ ] Horizontal links (citizens, stakeholders, municipalities)
- [ ] Vertical links (regional, national stakeholders)
- [ ] Cross-sector connections

### 4c. Transdisciplinary Approach
- [ ] Professionals from different fields involved
- [ ] Art + Science + Technology integration
- [ ] Non-academic knowledge incorporated
- [ ] Public engagement in knowledge creation

## 5. Additional Criteria (20/100 points)

### 5a. Results & Impact (10 pts)
- [ ] Clear measurable results documented
- [ ] Benefits to direct and indirect beneficiaries
- [ ] Transformative potential from global perspective

### 5b. Innovation (5 pts)
- [ ] Novel approach compared to business-as-usual
- [ ] Description of what makes it innovative

### 5c. Competitiveness / Financial Soundness (5 pts)
- [ ] Sound financial approach to the project
- [ ] Budget or sustainability plan

### 5d. Replicability (5 pts)
- [ ] Methodology can be transferred to other places
- [ ] Evidence of scalability or adaptation potential

## 6. Application Materials

- [ ] Project title and subtitle
- [ ] Project description (aligned with Category 3 language)
- [ ] Location and geographic scope
- [ ] Target audience/beneficiaries
- [ ] Timeline (start, milestones, completion)
- [ ] Team/collaborators list
- [ ] Visual documentation (photos, videos, diagrams)
- [ ] Evidence of results/impact (testimonials, data, press)
- [ ] Budget overview
- [ ] Links to project website/publications
